★あなたの親友がゾンビになった...あなたはどうする？👻★

最近見たゾンビ映画から着想を得てので、ゾンビにまつわるゲームを作ってみた。

ーソースコードー


    print("親友がゾンビになった...あなたはどうする？")

        

        # 最初の選択肢

    print("\n選択肢:")

    print("1. 助ける")

    print("2. 逃げる")

    

    choice1 = input("番号を入力してください (1/2): ")

    

    if choice1 == "1":

            # 親友を助ける選択肢

            print("\nあなたは親友を助ける方法を探し始める。")

            print("ネットでゾンビを治す薬を見つけるが、配送までに時間がかかり親友は完全にゾンビ化してしまった。")

            print("あなたはどうする？")

            

            # 2つ目の選択肢

            print("1. 再び逃げる")

            print("2. 親友との思い出が忘れられず、ゾンビ化しても一緒にいる")

            

            choice2 = input("番号を入力してください (1/2): ")

            

            if choice2 == "1":

                print("\n逃げることにした。")

                print("あなたの命は助かったが、ずっと後悔が残る。")

            elif choice2 == "2":

                print("\n一緒にいることに決めた。")

                print("ゾンビ化した親友に噛まれてしまう…。")

            else:

                print("\n無効な選択肢です。")

                

    elif choice1 == "2":

            # 親友から逃げる選択肢

            print("\nあなたは親友から逃げることにした。")

            print("しかし、逃げる途中で他のゾンビにも遭遇してしまい、ついに逃げられなくなる。")

            print("ゲームオーバー。")

            

        
    else:

            print("\n無効な選択肢です。ゲームオーバー。")
